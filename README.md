# Youth Soccer League Registration & Game Management Database (MySQL)

## Project Overview
This project is a relational database built in MySQL Workbench for a youth soccer league system. It is designed to manage important league data such as seasons, divisions, clubs, teams, registrations, and games.

This project includes the database model, SQL implementation, and a report showing the full development process.

## Tools Used
- MySQL Workbench
- SQL
- Relational Database Design
- ERD Modeling

## What I Did
- Designed the ERD for the database
- Created the database schema
- Built relational tables using primary keys and foreign keys
- Inserted sample data into the database
- Wrote SQL queries to test and analyze the data

## Database Name
`soccer_league_ps`

## Main Tables
- `SEASON_PS`
- `DIVISION_PS`
- `CLUB_PS`
- `TEAM_PS`
- `REGISTRATION_PS`
- `GAME_PS`

## Project Features
- Tracks league seasons and registration dates
- Organizes divisions by age group and player sex
- Stores club and team information
- Records registrations by division
- Stores game date, time, and ZIP code information
- Uses SQL queries to analyze league data

## Example Queries
This project includes SQL queries such as:
- Games per ZIP code
- Players per club
- Players per division in the current season
- Teams per coach
